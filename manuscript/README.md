# ESACRIF — Manuscript Package

This folder contains the manuscript and all manuscript-related material for the
ESACRIF (Explainable Survival-Aware Customer Retention Intelligence Framework)
Q1 journal submission. It is self-contained: the markdown documents, the
generated figures, and the generated data tables are all included here.

## Contents

| File | Description |
|---|---|
| `MANUSCRIPT.md` | Full journal manuscript (abstract → references → reproducibility appendix). |
| `SUBMISSION_PACKAGE.md` | Cover-letter / submission package, anticipated reviewer concerns, and responses. |
| `experimental_results.md` | Companion results document (per-experiment tables and findings). |
| `dataset_documentation.md` | Dataset metadata and provenance. |
| `figures/` | All 10 publication figures (PNG/PDF/SVG): Figures 1–10. |
| `tables/` | All 13 generated tables (CSV): Tables 1–11, 12, 12b. |

## Notes

- All numbers in `MANUSCRIPT.md` and `experimental_results.md` are taken directly
  from the CSVs in `tables/`.
- The figures are generated by `python pipeline.py` (seed 42) and written to
  `outputs/figures/` in the repository root; copies are mirrored here for convenience.
- The Figures/Table numbering matches the manuscript (Figure 10 = formal ESACRIF
  architecture; Table 12 = method comparison; Table 12b = statistical significance
  vs baseline models).
